Create `scripts/scrapers/aws-signature-v4.ts` implementing a minimal AWS Signature V4 signer (~100 lines). This replaces the 50MB AWS SDK for the 5-10 API calls we need.

The signer must:
1. Accept: HTTP method, full URL (with query string), credentials object ({ accessKeyId, secretAccessKey }), optional region (default: 'us-east-1'), optional service (default: 'pricing')
2. Build canonical request: method + URI (path-encoded) + query string (sorted) + signed headers (host;x-amz-date) + payload hash (empty for GET)
3. Build string to sign: algorithm + timestamp + credential scope + SHA-256(canonical request)
4. Derive signing key: HMAC-SHA256 chain (secret -> date -> region -> service -> 'aws4_request')
5. Calculate signature: HMAC-SHA256(signing_key, string_to_sign)
6. Return Headers object with: Host, X-Amz-Date, X-Amz-Content-Sha256, Authorization

Export:
- `signRequest(method: string, url: string, credentials: AwsCredentials, region?: string, service?: string): Headers`

Type:
- `export interface AwsCredentials { accessKeyId: string; secretAccessKey: string; }`

Security requirements (per RESEARCH.md Security Domain):
- NEVER log credentials (no console.log of accessKeyId or secretAccessKey)
- NEVER include credentials in error messages
- If signing fails, throw a generic Error without credential details

Use `node:crypto` for HMAC-SHA256:
```typescript
import { createHmac, createHash } from 'node:crypto';
```

Key implementation notes:
- The canonical URI must be URI-encoded path (/ is the root for the pricing API)
- The query string parameters must be sorted alphabetically by parameter name
- The timestamp format is ISO 8601 basic: `YYYYMMDDTHHmmssZ`
- The credential scope is `{date}/{region}/{service}/aws4_request`
- The signed headers are lowercase, semicolon-separated: `host;x-amz-content-sha256;x-amz-date`
- For GET requests, the payload hash is the SHA-256 of empty string: `e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855`

  private normalize(product: PricingProduct): NormalizedRecord | null {
    const attrs = product.product?.attributes;
    if (!attrs?.instanceType || !attrs?.location) return null;

    const price = this.extractOnDemandPrice(product);
    if (price === null) return null;

    return {

      timestamp: new Date().toISOString(),

    };
  }

  private extractOnDemandPrice(product: PricingProduct): number | null {
    const onDemand = product.terms?.OnDemand;
    if (!onDemand) return null;
    for (const offer of Object.values(onDemand)) {
      const dims = offer.priceDimensions;
      if (!dims) continue;
      for (const dim of Object.values(dims)) {
        const priceStr = dim.pricePerUnit?.USD;
        if (priceStr) return parseFloat(priceStr);
      }
    }
    return null;
  }
