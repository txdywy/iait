| Build config -> deployed static site | Vite base path and public asset handling determine what GitHub Pages serves. |
| Frontend bundle -> public users | All browser code and Vite public env values are visible to users. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Information Disclosure | `vite.config.ts`, `package.json` | mitigate | Do not introduce frontend secrets; only public static data is fetched in later plans. |
| T-03-02 | Denial of Service | `vite.config.ts` | mitigate | Set `base: '/iait/'` so static assets resolve on GitHub Pages. |
| T-03-03 | Tampering/XSS | `src/app/App.tsx` | mitigate | Render static shell copy through normal JSX text, no raw HTML injection. |
