<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Static GeoJSON/TopoJSON + index JSON -> MapLibre | Public static data controls map feature properties and marker labels. |
| Map click events -> Zustand/hash route | Untrusted browser events update UI state and shareable hash URLs. |
| Company data -> geographic overlay | Company data can be misrepresented as exact facility location if labels are wrong. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-10 | Tampering | `country-join.ts` | mitigate | Join via explicit ISO/name metadata and default unknown features to `hasComputeSignal: false` and `score: 0`. |
| T-03-11 | Denial of Service | `ComputeMap.tsx` | mitigate | Render loading/error states when geometry, latest index, or crossref queries are unavailable. |
| T-03-12 | Spoofing/Integrity | `drilldown.ts`, `ComputeMap.tsx` | mitigate | Company overlay strings must contain `corporate compute influence` or `AI CapEx signal` and `not an exact facility location`. |
| T-03-13 | Tampering | Hash route updates | mitigate | Only write hash routes from known `EntityType` values and selected IDs from static data/crossref. |
| T-03-14 | Information Disclosure | Map data | accept | The map displays public index data only; no private coordinates or secrets are introduced. |
</threat_model>
