<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser -> static JSON assets | Browser fetches public JSON/TopoJSON generated by pipeline or copied from mappings. |
| Static JSON -> typed UI contracts | Malformed or missing files can break rendering unless errors are explicit. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-04 | Denial of Service | `src/data/static-json.ts` | mitigate | Throw descriptive errors for non-OK static fetch responses and let consumers render error states. |
| T-03-05 | Tampering | `src/data/types.ts`, `src/data/queries.ts` | mitigate | Keep frontend contracts aligned with compiler outputs and add explicit derived cluster type instead of assuming cluster aggregate entities exist. |
| T-03-06 | Information Disclosure | `src/data/queries.ts` | accept | The plan fetches public static JSON only; no secrets or private data exist. |
